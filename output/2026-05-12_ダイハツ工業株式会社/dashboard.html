企業調査レポート（基本）: ダイハツ工業株式会社

調査日: 2026-05-12

1. 事業・経済基盤

1-1. ビジネスモデル

  • 顧客: 国内軽自動車ユーザー、アセアン小型車市場、トヨタ・マツダ・スバル（OEM供給先）
  • 提供価値: 「シンプル・スリム・コンパクト（SSC）」製造による低コスト・良品廉価なスモールカー
  • 収益モデル: フロー型（車両販売）。トヨタグループ完全子会社（2016年〜）
  • 2024年4月方針転換: 海外向け小型車開発をトヨタへ移管し、軽自動車専業メーカーへ回帰
  • 出典: ダイハツ公式・事業概要[1]、ダイハツ公式・今後の事業の方向性（2024年4月）[2]

1-2. 市場・競合

  • 業界内ポジション: 国内軽自動車市場シェア26.6%（2位）。認証不正前は17年連続首位
  • 主要競合: スズキ（首位・35.9%） / ホンダ（N-BOX等・約20%） / 日産
  • 回復状況: 2025年10月に22ヶ月ぶりに軽自動車メーカー首位に返り咲き（新型ムーヴが牽引）
  • 出典: 日経新聞・軽自動車首位陥落[3]、Car Watch 2025年10月販売ランキング[4]

1-4. 成長戦略

  • 軽自動車専業への回帰（2024年4月正式発表）
  • 開発・認証機能をトヨタコンパクトカーカンパニー配下に移行
  • 2027年度までにDX人材3,000人育成（2025年度末の1,000人目標を早期達成）
  • 電動化：2030年までに国内全商品を電動車化
  • 出典: ダイハツ公式・今後の方向性（2024年4月）[2]、プレスリリース DX人材3,000人育成[7]

5. 社会的インパクト・リスク

5-1. 不祥事・法令違反の履歴

認証試験不正問題（2023〜2024年）

  • 2023年4月に内部通報で発覚。第三者委員会調査により25試験項目・174件の不正が判明
  • 対象：64車種・3エンジン（最長30年以上の不正期間の可能性）
  • 不正内容：エアバッグ作動タイマーの不正操作、排ガス試験直前の触媒新品交換等
  • 国交省行政処分（2024年1月26日）：グランマックストラック等3車種の型式指定取消・是正命令
  • 経営責任：奥平総一郎社長・松林淳会長が2024年3月1日付で引責辞任
  • 真因（第三者委員会）：「短期開発を優先した経営判断」「物言えぬ組織風土」
  • 出典: 第三者委員会報告（2023年12月）[8]、行政処分（2024年1月）[9]

5-2. ESG・サステナビリティ

  • 「ダイハツグループ環境戦略2035」：2035年に生産カーボンニュートラル達成目標
  • 2030年までに国内全商品の電動車化
  • ISO 14001認証：国内全生産子会社が取得・更新済み
  • DAIHATSU Sustainability Report 2025を発行
  • 出典: ダイハツ公式・サステナビリティ[10]

5-3. 顧客評判

  • 認証不正による出荷停止で多数の納車遅延・キャンセルが発生。顧客信頼を大きく棄損
  • 2024年1月：キャスト・ピクシスジョイ 計32万2,740台のリコール
  • 2025年6月発売の新型ムーヴは先行予約で目標の2倍以上の受注、顧客信頼の回復に向かっている
  • 出典: ダイハツ公式・認証不正対応[11]、ベストカー ムーヴ販売首位[12]

2. 組織・カルチャー

2-1. 企業理念

  • スローガン: 「Light you up」（2017年、創立110周年に刷新）
  • ブランドビジョン: 「世界中の一人ひとりが自分らしく、軽やかに輝くモビリティライフを広げる」
  • 経営理念の5本柱: 顧客の笑顔と感動 / 相互の個性尊重と公平 / 地球と社会との共生 / スピード・ブレークスルー・率先垂範 / 世界一のスモールカーづくり
  • 出典: 企業理念｜ダイハツ工業[1]

2-2. 意思決定スタイル

  • 組織構造: トヨタ完全子会社。2024年4月から開発・認証機能をトヨタコンパクトカーカンパニー傘下に移行
  • スタイル: トップダウン傾向。「年功序列の大企業文化が根強い」との口コミ多数
  • 一方で「管理職の面倒見が良く話しかけやすい」「困ったとき助けを求めると聞いてくれる」という声も
  • 認証不正の根本原因として第三者委員会が「物言えぬ文化」を指摘。現在改革途上

2-3. 評価制度

  • 基本方針: 機会均等（職種・学歴・年齢に関係なくチャレンジできる）＋実力主義
  • 理念・ビジョン・方針の達成に向けた行動と成果を評価
  • 評価者教育（上司向け）と本人・上司によるフィードバック面談を実施
  • 実態: 年功序列の要素が強く勤続年数で基本給が上昇。「仕事のレベルは上がるが昇格しない」との声
  • 出典: 人事制度｜ダイハツ キャリア採用[2]

2-4. 人的構成

  • 従業員数（単体）: 約11,154〜12,470人
  • グループ全体: 約46,191人（2023年3月時点）
  • 平均年齢: 38.3〜41.2歳（情報源により差異あり）
  • 平均勤続年数: 17.1年（男性19.5年、女性15.9年）→定着率の高さを示す
  • 出典: 年収ガイド ダイハツ工業[3]

3. 職務・スキル

3-1. 業務範囲・チーム

  • 軽自動車の設計・開発・製造・品質管理が中核
  • 中途採用の主な募集職種：開発技術者、品質問題解析、メカ設計エンジニア、人事・企画・DX推進等
  • DX推進チームを各部署に6〜9人配置することが目標
  • 出典: ダイハツ キャリア採用 募集職種[4]

3-2. 技術スタック

  • 製造: SSC（シンプル・スリム・コンパクト）製造方式
  • デジタル: 2022年から生成AI活用開始、2023年に独自生成AIを運用開始
  • 開発インフラ: 認証不正後に開発・認証プロセスの全面見直しを実施
  • AI活用: 将来目標「社員の9割以上が日常的に生成AIを活用」
  • 出典: MONOist ダイハツ全社デジタル改革[5]

3-3. 教育・研修

  • 体系: OJT中心に実習・研修プログラムを組み合わせ。100講座以上の部門別スキルアップ研修
  • 新入社員: 会社基礎・社会人基礎力・安全基礎・製造工程・基本訓練・実車組付けなど
  • DX人材育成: 2025年度末目標1,000人を早期達成。次フェーズは2027年度までに3,000人
  • 出典: 人材育成｜ダイハツ新卒採用[6]

3-4. キャリアパス

  • 資格キャリア制度あり。職種間異動（営業→企画、サービス→営業等）が可能
  • 懸念: 「会社ありきのスキルしか身につかない」「外に出たら何もできない」という口コミも
  • 「仕事のレベルは上がるが昇格しない」という年功序列的な評価への不満もあり

4. 労働条件・環境

4-1. 労働時間

  • 所定労働時間: 標準的な8時間（年間休日120日以上）
  • 残業: 月平均26.4〜27.3時間。部署により差が大きく30〜40時間/月の部署も
  • フレックス: 取得しやすい環境との評価。年休も遠慮なく取れる

4-2. リモートワーク

  • 2020年2月から導入。スタッフ職（約6,400人・国内従業員の約半数）が対象
  • 開発職はリモートワークを取り入れやすい環境
  • 副業・兼業人材は「週1回程度勤務でリモートワーク想定、月1回程度出社」体制

4-3. 福利厚生・育休・副業

  • 福利厚生: 退職金・確定拠出型年金、家族手当、保養施設（ラコンテ有馬）、社内クラブ・グラウンド
  • 有給休暇: 消化率88.9%（業界平均70.9%を大幅上回る）
  • 育休: 取得しやすい環境との口コミ多数。男性育休も広まりつつある。育休後復帰率高い
  • 副業: 解禁を検討・推進（自動車メーカーとして先進的。外部副業・兼業人材の受け入れ制度も整備）

4-4. 給与

  • 平均年収: 651万円（有価証券報告書ベース）。業界平均629万円より約22万円高い
  • 年代別: 20代478万円 / 30代573万円 / 40代669万円 / 50代720万円（推定）
  • 賞与: 年2回（7月・12月）。年間約5〜6ヶ月分
  • 残業代: 1分単位で支給（サービス残業はほぼなし）
  • 出典: 年収ガイド・ダイハツ工業[3]、CareerMine・ダイハツ工業年収[7]
