## 1. 事業・経済基盤（継続性の確認）

### 1-1. ビジネスモデル
- **誰に**: アーティスト・音楽レーベル・広告主・音楽ファン（BtoB＋BtoC）
- **何を**: 音楽を切り口とした総合エンタテインメントサービス
- **どのように**: 3セグメントのハイブリッド型
  - **メディア事業**: 音楽専門チャンネル「スペースシャワーTV」の放送・番組制作・広告販売、オンデマンド配信
  - **ライブ・コンテンツ事業**: イベント企画運営、ライブハウス運営、アーティストマネジメント、レーベル運営、原盤企画制作、出版
  - **ソリューション事業**: 音楽配信ディストリビューション（SPACE SHOWER FUGA）、デジタルマーケティング、EC・マーチャンダイズ、ファンクラブ運営（SKIYAKI）
- **グループ構成**: スペースシャワーSKIYAKIホールディングス（東証スタンダード4838）の100%子会社。グループ会社にSKIYAKI、インフィニア、SPACE SHOWER FUGA等
- **出典**: [事業概要（SSSK-HD）](https://sssk-hd.com/contents/business-summary) / [Wikipedia](https://ja.wikipedia.org/wiki/%E3%82%B9%E3%83%9A%E3%83%BC%E3%82%B9%E3%82%B7%E3%83%A3%E3%83%AF%E3%83%BC%E3%83%8D%E3%83%83%E3%83%88%E3%83%AF%E3%83%BC%E3%82%AF)

### 1-2. 市場シェアと競合
- **業界内ポジション**: テレビ業界の売上高ランキング16位/29社（2023年3月決算）。音楽専門チャンネルとしては国内最大級
- **主要競合**:
  - **MTVジャパン** — グローバルブランド力が強み。ただし日本市場での存在感は縮小傾向
  - **WOWOW** — 音楽以外のコンテンツ（映画・スポーツ）も含む総合エンタメ。規模で上回る
  - **エイベックス** — アーティストマネジメント・レーベル・ライブ事業で重複。規模・知名度で上回る
- **対象企業の強み**: 
  - 音楽特化の「メディア×ライブ×デジタル」の垂直統合モデル
  - 伊藤忠・フジメディア・KDDIの大株主による強固な資本基盤と事業シナジー
  - FUGA社との合弁によるグローバル音楽配信ネットワーク
- **対象企業の弱み**: 
  - テレビ放送（メディア事業）は構造的な衰退トレンド。有料放送加入者数の減少リスク
  - ライブ事業は天候・疫病等の外部要因に左右されやすい
- **出典**: [業界Search](https://gyokai-search.com/kigyou/spaceshower.html) / [Branc 決算分析](https://branc.jp/article/2023/08/04/670.html)

### 1-4. 成長戦略
- **中期経営計画「Ignite 2027」**（2026年3月期〜2028年3月期）: 2024年11月発表。最終年度（2028年3月期）の定量目標:
  - 売上高 **240億円**
  - 営業利益 **16億円**（営業利益率 **7%**）
  - EBITDA **22億円**
  - ROE **10%超**
  - 3年間累計営業CF **44億円**
- **セグメント再編**: 「メディア」「ライブ・コンテンツ」「ソリューション」の3セグメント体制。成長の重点領域は**ライブ・コンテンツ**と**ソリューション**
- **SKIYAKI経営統合（2024年4月）**: ファンクラブ・EC・マーチャンダイズのプラットフォーム機能を取り込み、アーティストへの360°ソリューション提供体制を構築
- **グローバル展開**: オランダのFUGA社と合弁会社（SPACE SHOWER FUGA）を設立。日本の音楽コンテンツの海外配信を強化
- **DX推進**: 既存コンテンツのDXによる付加価値向上、デジタルマーケティング機能の強化
- **出典**: [中期経営計画「Ignite 2027」発表リリース](https://sssk-hd.com/contents/866407) / [事業概要（SSSK-HD）](https://sssk-hd.com/contents/business-summary) / [PR TIMES FUGA合弁](https://prtimes.jp/main/html/rd/p/000001673.000002929.html)

### 5-2. サステナビリティ・ESG
- **取り組み概要**: 中期経営計画にサステナビリティ対応を明記。具体的な取り組みあり
- **多様性推進**:
  - 女性管理職比率25%（2023年度）
  - 出産後の復職率100%（過去5年間）
  - 障がい者雇用: 屋内農園型雇用支援サービスを活用したハーブティー製造
- **働き方改革**: ハイブリッド勤務・フレックス勤務の導入
- **音楽文化の持続的発展**: 多様性の実現、アーティストの自由で独立的な活動支援を企業理念に据える

### 5-3. 顧客からの評判
- **サービスの評価**: 「スペースシャワーTV」は音楽ファンから長年支持されるブランド。1989年開局の歴史を持つ。スカパー！・J:COMほか主要プラットフォームで継続配信中
- **業界評価**: コロナ禍からの収益力回復が業界メディア（Branc等）で評価されている
- **社員評価（OpenWork / キャリコネ / 転職会議）**:
  - 【良い点】音楽好きが集まるため社員間コミュニケーションが活発。年間休日130日＋リフレッシュ休暇5日。産休・育休取得実績あり（復職率100%）。部署によっては女性管理職が多く働きやすいとの声
  - 【懸念点】近年は本部ごとの縦割り組織化が進み、横断連携・横串プロジェクトが薄れているとの指摘あり。昇進・昇給は年功序列の色が残っており、20代の給与水準がやや低め。直属上司との相性に依存しやすい構造
- **出典**: [Branc 決算分析](https://branc.jp/article/2023/08/04/670.html) / [OpenWork](https://www.openwork.jp/company.php?m_id=a0910000000Fr9z) / [キャリコネ](https://careerconnection.jp/review/2924/kutikomi/) / [転職会議](https://jobtalk.jp/companies/1584)
