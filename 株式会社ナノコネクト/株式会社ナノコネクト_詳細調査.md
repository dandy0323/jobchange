## 2. 組織・カルチャー（相性の確認）

### 2-1. 企業理念（ミッション・ビジョン・バリュー）
- **公式の理念**: 公式サイトに「ミッションと4つの大切にしていること（バリュー）」の記載あり。ただし具体的な文言はWebFetchでは取得できなかった
- **現場への浸透度**: 
  - 組織戦略として「自律的意思決定ができて成果を出せる組織」の構築を掲げている
  - 「社員は友達ではなく同僚という距離感」を重視し、客観的根拠に基づく提案文化が根付いている
  - 個人のわがままではなく「組織向上につながる案」が求められる風土
- **出典**: [Green 企業ページ](https://www.green-japan.com/company/4716) / [ナノコネ会社概要](https://www.nanoconnect.co.jp/about/corporation/)

### 2-2. 意思決定スタイル
- **組織構造**: フラット型。風通しの良さは口コミサイトで高評価（★4.0〜4.1）
- **意思決定の傾向**: ボトムアップ寄り。「自律的意思決定」を組織戦略として掲げており、現場からの提案を重視。ただし「部署によるところが大きい」との指摘もあり
- **出典**: [エン カイシャの評判](https://en-hyouban.com/company/10100600767/) / [Green 企業ページ](https://www.green-japan.com/company/4716)

### 2-3. 評価制度
- **評価体系**: 成果主義ベース。年功序列ではなく、技術力＋基本的行動（遅刻しない等）の両面で評価
- **昇給**: 年2回
- **昇進機会**: 年4回（業界では多い方）
- **課題**: 
  - 客先常駐（SES）の場合、チーム構成に昇進が左右されるとの口コミあり
  - 「実力や評価よりもチーム構成に依存する」ケースが報告されている
- **良い点**: 評価前に上司と面談があり、「対等で和やかな雰囲気」でキャリアについて話せる
- **出典**: [就活会議](https://syukatsu-kaigi.jp/companies/129511) / [転職会議](https://jobtalk.jp/companies/7662)

### 2-4. 組織の人的構成
- **従業員数**: 190名（役員・内定者含む）＋パートナー27名 = 計217名（2026年2月時点）
- **平均年齢**: 31.3歳（2022年2月時点）
- **離職率**: 公式数値は非公開。口コミでは「若手の離職率の高さ」が指摘されており、中間層が少ない傾向との声あり
- **男女比**: 女性エンジニアが多いとの記載あり
- **出典**: [ナノコネ会社概要](https://www.nanoconnect.co.jp/about/corporation/) / [doda企業ページ](https://doda.jp/DodaFront/View/Company/j_id__10100600767/)

## 3. 職務・スキル（キャリア形成の確認）

### 3-1. 具体的な業務範囲
- **担当フェーズ**: 企画〜運用保守まで一貫対応。「超上流工程」（ビジネスモデル構築・企画提案）から携われるのが特徴
- **業務タイプ**: 
  - **自社開発**: AI LinKa、Skywalker、みえるーむ等の自社プロダクト開発
  - **受託開発**: クライアント向けWebシステム・アプリ開発
  - **SES**: 客先常駐型の開発（配属先による業務内容の差が大きい）
- **チーム構成**: 具体的な情報なし。募集要項では「チーム開発経験」が応募条件
- **注意点**: 求人では「常駐なし」と明記されているが、口コミではSES（客先常駐）に関する言及が多数あり、矛盾が見られる → 面接で要確認
- **出典**: [ナノコネ中途採用ページ](https://www.nanoconnect.co.jp/recruit-career/engineer/) / [転職会議](https://jobtalk.jp/companies/7662)

### 3-2. 技術スタック・使用ツール
- **言語・フレームワーク**: PHP, JavaScript, Java, C#, VB, Kotlin, Swift, DB言語
- **インフラ・クラウド**: 情報なし（面接で確認推奨）
- **AI活用状況**: 
  - 自社でAIプロダクト（LinKa, AI Seren）を開発・運用
  - 生成AIを「みえるーむ」に搭載済み
  - 機械学習を活用した「本物のAI」開発を標榜
- **開発手法**: 情報なし
- **技術領域**: AI、ゲーミフィケーション、データマイニング、IoT、Fintech、O2O、SaaS、位置情報ソリューション、組込みシステム
- **出典**: [ナノコネ中途採用ページ](https://www.nanoconnect.co.jp/recruit-career/engineer/) / [ナノコネ公式サイト](https://www.nanoconnect.co.jp/)

### 3-3. 教育・研修体制
- **研修制度**: 入社後のOFF-JT（座学研修）が整備されている。「未経験でもやる気と努力次第で活躍できる」との口コミ
- **自己研鑽支援**: キャリア支援制度あり（詳細は非公開）
- **社内制度**: 「自社製品企画制度」があり、社員がプロダクト企画に参加できる
- **技術発信**: Developers Summit 2018〜2022年で4年連続ベストスピーカー賞受賞。技術発信を重視する文化
- **出典**: [Green インタビュー](https://www.green-japan.com/company/4716/interview) / [Green 企業ページ](https://www.green-japan.com/company/4716)

### 3-4. キャリアパス
- **社内キャリアの選択肢**: 
  - 昇進機会が年4回あり、成果主義で早期のキャリアアップが可能
  - 自社プロダクト企画への参画チャンスあり
- **課題**: 「キャリアパスが不透明」との口コミあり。中間層が少なく、モデルケースが見えにくい可能性
- **出典**: [IT転職800](https://it-jobnavi.com/nanoconnect-yabai/) / [就活会議](https://syukatsu-kaigi.jp/companies/129511)

## 4. 労働条件・環境（生活の質の確認）

### 4-1. 労働時間
- **所定労働時間**: 8時間/日
- **フレックス制度**: あり。出社7:00〜11:00の間で自由、コアタイム11:00〜14:00
- **平均残業時間**: 月19〜28.5時間（データソースにより幅あり）
- **懸念点**: 本社勤務は裁量労働制。口コミでは「20時間分の固定残業代で30〜40時間働かされる」「22時まで毎日いる人も多い」との声あり
- **出典**: [ナノコネ中途採用ページ](https://www.nanoconnect.co.jp/recruit-career/engineer/) / [転職会議 残業口コミ](https://jobtalk.jp/companies/7662/answers?topic=%E6%AE%8B%E6%A5%AD)

### 4-2. リモートワーク
- **制度の有無**: あり。テレワーク手当も支給
- **実態**: ローテーション出社とハイブリッドワーク（テレワーク＋出社）を自由に選択可能。神戸本社または自宅からのテレワーク対応
- **口コミ評価**: 「リモートワーク導入でワークライフバランスが一気に高まった」と好評
- **出典**: [ナノコネ中途採用ページ](https://www.nanoconnect.co.jp/recruit-career/engineer/) / [エンゲージ会社の評判](https://en-hyouban.com/company/10100600767/18/)

### 4-3. 福利厚生
- **主な制度**:
  - お昼寝制度（毎日13時に全社消灯、15分のパワーナップ）
  - 短時間勤務制度（4時間・6時間勤務が選択可。男性も取得可能）
  - 産休育休制度（女性取得率100%）
  - 家族手当、住宅手当、常駐手当、テレワーク手当
  - 交通費全額支給
  - 社会保険完備
  - 自社製品企画制度、キャリア支援制度
  - 軽音楽部等のクラブ活動
- **育休取得実績**: 女性取得率100%。男性の取得実績は情報なし
- **副業可否**: 情報なし
- **出典**: [ナノコネ中途採用ページ](https://www.nanoconnect.co.jp/recruit-career/engineer/) / [Green 企業ページ](https://www.green-japan.com/company/4716)

### 4-4. 給与
- **想定年収レンジ**: 
  - エンジニア（中途）: 457万〜800万円（月30.5万〜53.5万円）
  - 口コミ平均年収: 437万〜480万円（回答者平均年齢33.3歳）
  - 年収幅: 240万〜850万円
- **給与構成**:
  - 基本給 ＋ 固定残業代（裁量労働制で20時間分）
  - 賞与: 年2回（実績3.5ヶ月分）
  - 昇給: 年2回
- **懸念点**: 「基本給が低く、手当で補填する構造」「給与面の不満が複数サイトで最多の口コミ」との指摘あり
- **出典**: [ナノコネ中途採用ページ](https://www.nanoconnect.co.jp/recruit-career/engineer/) / [エン カイシャの評判 年収](https://en-hyouban.com/company/10100600767/salary/) / [就活会議 年収口コミ](https://syukatsu-kaigi.jp/companies/129511/post_items/2/word_mouths)
