## 2. 組織・カルチャー（相性の確認）

### 2-1. 企業理念（ミッション・ビジョン・バリュー）

- **ミッション（公式）**:
  > *"驚きを あなたのもとへ"*
  「驚き」は相手の期待を上回って初めて生まれるものとし、常に成長し一歩先を歩むことを求める理念。

- **ビジョン（公式）**:
  1. *"小さなつながりである、目の前の「あなた」を大事にする"*
  2. *"多くの人にすごい！と言わせる「驚き」を与えていきたい"*
  - 社名の語源: **nano（10億分の1）＋ connection（つながり）** = 小さなつながりを大切にする

- **コアバリュー（4つ）**:
  1. **価値創造への献身** — 相手の期待を上回る成果を生み出す姿勢
  2. **競争力の強化** — 個人の専門性を継続的に磨き社会に活かす
  3. **課題解決思考** — 不可能とされた問題に知恵を絞り続ける実行力
  4. **自己責任の原則** — 結果は過去の選択に由来すると認識し自己成長を追求

- **現場への浸透度**: 採用ページ・求人票でも理念を前面に出しており、選考でも重視される。OpenWorkの「社員の士気」スコアは3.2点と中程度で、理念への共感度は社員によって差あり。

- **出典**: [経営理念 | ナノコネ](https://www.nanoconnect.co.jp/about/philosophy/) / [OpenWork](https://www.openwork.jp/company.php?m_id=a0C1000000iusgV)

### 2-2. 意思決定スタイル

- **組織構造**: フラット志向。「自律的意思決定ができて成果を出せる組織をつくる」を組織戦略として掲げる
- **意思決定の傾向**: 社員が改善案・業務効率化案を提案できる文化。「個人のわがままではなく、客観的に組織の向上につながる案」が条件。通った提案は評価され、ポジション・所得向上に直結
- **人間関係の方針**: 「社員は『友達』ではなく『同僚』という適切な距離感をキープ」と明示（プロフェッショナルな関係性を重視）
- **懸念点**: 「社長の好き嫌いで人事評価が変わる傾向がある」との口コミが複数あり、トップダウンの側面も
- **出典**: [Green 企業情報](https://www.green-japan.com/company/4716) / [転職会議](https://jobtalk.jp/companies/7662)

### 2-3. 評価制度

- **評価体系**: 成果主義（年功序列ではない）。OpenWork 実力主義スコア **3.5点**
- **昇給**: 年2回（同一役職内で半年に1度）
- **昇進機会**: 年4回（業界平均より多い）
- **昇進基準**: 技術力に加え「遅刻しない等の社会人として最低限の行動」も必須条件
- **否定的口コミ**: 「社長の好き嫌いで人事評価が変わる」「評価基準が不透明」— OpenWork 人事評価の適正感スコア **3.2点**
- **出典**: [OpenWork 年収・給与制度](https://www.openwork.jp/company_answer.php?m_id=a0C1000000iusgV&q_no=2) / [転職会議](https://jobtalk.jp/companies/7662)

## 3. 職務・スキル（キャリア形成の確認）

### 3-1. 具体的な業務範囲

- **担当フェーズ**: ビジネスモデル構築の「超上流工程」（構想・企画）から設計・開発・運用保守まで一括受託が強み
- **業務の特徴**: 直取引主義のため、エンジニアが顧客と直接折衝する機会が多い。サブリーダー以上になると顧客折衝が増加
- **開発領域**: Webシステム・業務システム / スマホアプリ（Android / iOS）/ 組込みシステム（ECU・ナビゲーション等）/ クラウドシステム / AI・機械学習・ディープラーニング / IoT研究
- **出典**: [エンジニア募集 | ナノコネ](https://www.nanoconnect.co.jp/recruit-career/engineer/) / [Green エンジニア求人](https://www.green-japan.com/company/4716/job/168471)

### 3-2. 技術スタック・使用ツール

- **言語**: PHP、JavaScript、Java、C#、VB、Kotlin（Android）、Swift（iOS）、SQL系

- **自社プロダクト**:
  | プロダクト | 概要 |
  |-----------|------|
  | **LinKa** | 対話型AI接客ツール、1,000超の導入実績 |
  | **Sky Walker** | ゲーミフィケーションパッケージ |
  | **Seren（旧DME）** | ビッグデータマイニングエンジン |

- **インフラ・クラウド**: 具体的なサービス名は公開情報では確認できず
- **AI活用状況**: LinKaを中心に実用レベルのAI開発実績あり。AI・IoT研究チームが全社員公募で活動中
- **スキル評価基準**: ETSS（組込みスキル標準）、CMMI、PMBOKを活用
- **出典**: [エンジニア募集 | ナノコネ](https://www.nanoconnect.co.jp/recruit-career/engineer/) / [様々な取り組み | ナノコネ](https://www.nanoconnect.co.jp/about/action/)

## 4. 労働条件・環境（生活の質の確認）

### 4-1. 労働時間

| 項目 | 詳細 |
|------|------|
| 所定労働時間 | 1日8時間 |
| 勤務体制 | フレックスタイム制（出社時間帯 7:00〜11:00、コアタイム 11:00〜14:00） |
| 年間休日 | 125日（完全週休2日制） |
| 月平均残業時間 | **17.3〜23.1時間**（各口コミサイト集計値） |

- **裁量労働制**: 社内勤務者は適用。「1日1.25時間分の残業代が確実に支給される」との口コミあり
- **懸念点**: 「月間160時間超の残業があった」（退職済み男性SE、2024年10月）の極端な事例も。案件・部署によって残業格差あり
- **出典**: [転職会議 残業口コミ](https://jobtalk.jp/companies/7662/answers?topic=%E6%AE%8B%E6%A5%AD) / [OpenWork ワークライフバランス](https://www.openwork.jp/company_answer.php?m_id=a0C1000000iusgV&q_no=6)

### 4-2. リモートワーク

- **制度の有無**: あり。「常駐勤務なし」を中途採用求人に明記
- **実態**: 神戸本社勤務 または 自宅テレワークを選択可能。ローテーション出社・ハイブリッドも自由選択。テレワーク手当あり
- **短時間勤務**: 育児等の事由で4時間・6時間・通常勤務から選択可
- **懸念点**: 「テレワーク時にZoomで常時監視される状況」（退職済み女性開発職、2022年5月）— 現状は変更されている可能性あり、面接で確認推奨
- **出典**: [エンジニア募集 | ナノコネ](https://www.nanoconnect.co.jp/recruit-career/engineer/) / [エン カイシャ 働き方](https://en-hyouban.com/company/10100600767/18/)
